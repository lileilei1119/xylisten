### 留声
留声是一款使用TTS语音播报您个人笔记的APP，内置了简洁实用的富文本编辑器。基于flutter，支持android和iOS

### 功能
- 富文本笔记的新建、编辑等功能
- 对接tts（我已经将tts单独封装成了一个插件，支持android和iOS）
- 智能抓取网页功能
- 播放器控件（含迷你版 和 全屏版）
